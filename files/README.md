# Marketing ROI Calculator

A real-time marketing performance calculator for three business models:
- **eCommerce** — ROAS, True ROAS, CPA, profit & lifetime profit
- **Lead Generation** — CPL, CPA, close rate, ROI
- **EdTech / SaaS** — MRR, LTV, trial conversion, lifetime profit

Pure HTML — no dependencies, no build step, works offline.

---

## Deploy to Vercel in 3 steps

### 1. Push to GitHub

```bash
git init
git add .
git commit -m "first commit"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/marketing-calculator.git
git push -u origin main
```

### 2. Connect to Vercel

1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Import your GitHub repo `marketing-calculator`
3. Framework preset: **Other**
4. Click **Deploy**

Done — Vercel gives you a live URL like `marketing-calculator.vercel.app`

### 3. Use offline

Open `index.html` directly in any browser — no server needed.

---

## Customise

All logic is in the `<script>` block at the bottom of `index.html`.  
All styling is in the `<style>` block at the top.  
No frameworks, no build tools — just edit and save.
